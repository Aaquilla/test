1. Що таке CSS?

  • a) Cascading Style Sheets
  • b) Computer Style Sheets
  • c) Colorful Style Sheets
  • d) Creative Style Sheets
Відповідь: a) Cascading Style Sheets

2. Яке правило використовується для задання кольору тексту в CSS?

  • a) color
  • b) text-color
  • c) font-color
  • d) text
Відповідь: a) color

3. Яке правило використовується для задання розміру шрифту в CSS?

  • a) size
  • b) font-size
  • c) text-size
  • d) font
Відповідь: b) font-size

4. Яке правило використовується для задання кольору фону в CSS?

  • a) bg-color
  • b) background-color
  • c) color
  • d) background
Відповідь: b) background-color

5. Як можна задати стиль рамки для елемента в CSS?

  • a) border-style
  • b) border
  • c) border-width
  • d) border-color
Відповідь: b) border

6. Як можна задати стиль тексту для елемента в CSS?

  • a) font-style
  • b) text-style
  • c) style
  • d) font
Відповідь: a) font-style

  • a) margin

9. Яке правило використовується для задання внутрішнього відступу між елементами в CSS?
