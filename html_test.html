1. Що таке HTML?

  • a) Hyper Text Markup Language
  • b) Hyperlinks and Text Markup Language
  • c) Home Tool Markup Language
  • d) Hyperlinks and Text Makeup Language
Відповідь: a) Hyper Text Markup Language

2. Яка тегова пара використовується для створення таблиці в HTML?

  • a) <table>\</table>
  • b) <div>\</div>
  • c) <section>\</section>
  • d) <form>\</form>
Відповідь: a) <table>\</table>

3. Які теги використовуються для створення нумерованого списку в HTML?

  • a) <ol>\</ol>
  • b) <ul>\</ul>
  • c) <li>\</li>
  • d) <nl>\</nl>
Відповідь: a) <ol>\</ol>

4. Який елемент використовується для створення гіперпосилання в HTML?

  • a) <a>\</a>
  • b) <link>\</link>
  • c) <href>\</href>
  • d) <ref>\</ref>
Відповідь: a) <a>\</a>

5. Які теги використовуються для створення безпорядкованого списку в HTML?

  • a) <ol>\</ol>
  • b) <ul>\</ul>
  • c) <li>\</li>
  • d) <nl>\</nl>
Відповідь: b) <ul>\</ul>

6. Якими символами огороджуються атрибути в HTML?

  • a) ""
  • b) {}
  • c) <>
  • d) ''
Відповідь: a) ""

7. Який тег використовується для створення зображення в HTML?

  • a) <img>\</img>
  • b) <image>\</image>
  • c) <picture>\</picture>
  • d) <src>\</src>
Відповідь: a) <img>\</img>

8. Яким елементом ви починаєте кожну HTML-сторінку?

  • a) <head>\</head>
  • b) <body>\</body>
  • c) <title>\</title>
  • d) <html>\</html>
Відповідь: d) <html>\</html>

9. Що означає абревіатура URL?

  • a) Universal Resource Locator
  • b) Uniform Resource Link
  • c) Unified Resource Language
  • d) Universal Reference Link
Відповідь: a) Universal Resource Locator

10. Яким елементом ми визначаємо заголовок сторінки в HTML?

  • a) <head>\</head>
  • b) <body>\</body>
  • c) <title>\</title>
  • d) <html>\</html>
Відповідь: c) <title>\</title>

11. Як ви можете вказати коментар в HTML?

  • a) <!-- це коментар -->
  • b) // це коментар
  • c) /* це коментар */
  • d) <comment>це коментар</comment>
Відповідь: a) <!-- це коментар -->

13. Які теги використовуються для створення заголовків у HTML?

  • a) <header>\</header>
  • b) <h1>\</h1>
  • c) <heading>\</heading>
  • d) <head>\</head>
Відповідь: b) <h1>\</h1>

14. Що таке елемент <div> в HTML?

  • a) Тег для визначення таблиці
  • b) Тег для визначення контейнера для елементів блоку
  • c) Тег для визначення посилання
  • d) Тег для визначення зображення

15. Які теги використовуються для створення посилання в HTML?

  • a) <a>\</a>

  • c) <href>\</href>
